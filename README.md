# Coding test

## Created with React and Express

## Instructions to run
    $ git clone https://github.com/asenchekov/code-test-react-express.git
    $ cd code-test-react-express
    $ cd front-end
    $ npm i
    $ npm run build
    $ cd ../back-end
    $ npm i
    $ npm start

    // app started at http://localhost:8080/
